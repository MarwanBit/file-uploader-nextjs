    Function SharedFolderTree

    • SharedFolderTree(): Element

      Tree component for displaying shared folder hierarchy.

      Fetches and displays the complete folder tree structure from the FolderContext. Supports recursive rendering of nested folders and files. Highlights the currently active folder based on FolderContext state.

      Returns Element

      Interactive tree structure of folders and files

      Example

      <SharedFolderTree />

      Example

      // In a shared folder sidebar
      <SharedAppSidebar>
        <SharedFolderTree />
      </SharedAppSidebar>

      Remarks

      • Uses rootFolder from FolderContext (useFolder hook)
      • Highlights active folder with blue background
      • Files are displayed as non-clickable leaf nodes
      • Folders are collapsible/expandable
      • Clicking folder updates currentFolder in FolderContext
      • No URL changes (navigation managed by context)
      • Read-only: designed for shared folder views

      See

      • useFolder for the folder context hook
      • FolderTree for the regular (non-shared) version
