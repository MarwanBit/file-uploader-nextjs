      Function

      Uploads a file to a specific folder.

      This endpoint handles file uploads to a designated folder. The file is uploaded to S3 storage and its metadata is saved in the database. The file is associated with both the target folder and the user's root folder.

      Parameters

      • request: Request

        The incoming HTTP request object

        • body

          FormData containing the file to upload

          // FormData structure:
          formData.append('file', fileObject);
      • params: { params: Promise<{ folderId: string }> }

        Route parameters

      Returns Promise<Response>

      A Response object containing a success message

      Async

      POST

      Throws

      Returns 401 if the user is not authenticated

      Throws

      Returns 500 if root/current folder not found or upload fails

      Example

      // Upload a file to a folder
      const formData = new FormData();
      formData.append('file', fileInput.files[0]);

      const response = await fetch('/api/folders/folder-123/files', {
        method: 'POST',
        body: formData
      });

      const result = await response.json();
      console.log(result.message); // "Worked as expected!"

      Example

      // Complete upload handler with progress
      async function uploadFile(folderId: string, file: File) {
        const formData = new FormData();
        formData.append('file', file);
        
        try {
          const response = await fetch(`/api/folders/${folderId}/files`, {
            method: 'POST',
            body: formData
          });
          
          if (!response.ok) {
            throw new Error('Upload failed');
          }
          
          const result = await response.json();
          console.log('Upload successful:', result);
          return result;
        } catch (error) {
          console.error('Upload error:', error);
          throw error;
        }
      }

      Example

      // Multiple file upload
      const files = Array.from(fileInput.files);
      for (const file of files) {
        const formData = new FormData();
        formData.append('file', file);
        
        await fetch(`/api/folders/${folderId}/files`, {
          method: 'POST',
          body: formData
        });
      }

      Remarks

      • Requires valid Clerk authentication
      • Accepts multipart/form-data with a 'file' field
      • File is stored in S3 with a structured key path
      • Creates database record with file metadata
      • Uses FolderService.uploadFileToFolder internally
      • The file buffer is read into memory before upload

      See

      • FolderService.uploadFileToFolder for the underlying implementation
      • FolderService.getFolder for folder retrieval
      • DELETE /api/files/[id] for deleting uploaded files
